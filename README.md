# Todoist Minimal Plugin

## Overview

The **Todoist Minimal** plugin automatically creates tasks in Todoist using the provided API key, found at Settings > Integrations > Developer.

This plugin can be used as is, but isn't fully featured and is more of an example what could be done. Please create a fork if you would like more features :)

## Features

Automatically create tasks in your Todoist account

## Installation

**Todoist Minimal** is a `Vocode Community Plugin`

## Configuration

Once the plugin is added, configure it with the following options:

#### API Token

Specify the Todoist API Token to use to interact with the [official API](https://developer.todoist.com/api/v1)

## Usage Example

1. **API Token**: `0123456789abcdef0123456789abcdef01234567`

When a transcription is processed, the plugin will create a task with the text entered/transcribed and a label of the timestamp created at on Todoist in the associated account.
